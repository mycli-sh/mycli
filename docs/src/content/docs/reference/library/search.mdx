---
title: my library search
description: Search public libraries in the registry.
---

```bash
my library search <query>
```

Searches the registry for public libraries matching the given query.

## Arguments

| Argument | Description |
|----------|-------------|
| `query` | Search query (required) |

## Examples

```bash
# Search for kubernetes-related libraries
my library search kubernetes

# Search for deployment tools
my library search deploy
```
