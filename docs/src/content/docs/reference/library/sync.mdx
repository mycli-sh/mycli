---
title: my library sync
description: Sync registry libraries with the server.
---

```bash
my library sync
```

Fetches the latest command catalog from the server for all registry-installed libraries. This bypasses the ETag cache to ensure you have the most up-to-date commands.

Requires login (`my cli login`).

## Examples

```bash
# Sync all registry libraries
my library sync
```
