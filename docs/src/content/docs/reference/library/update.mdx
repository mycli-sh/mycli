title: my library update (deprecated)

---

Library updates are now handled differently depending on how the library was installed:

## Git-backed sources

Use [`my source update`](/reference/source/update/) to pull the latest changes from git-backed source repositories:

```bash
# Update all sources
my source update

# Update a specific source
my source update my-tools
```

## Registry libraries

Use [`my library sync`](/reference/library/sync/) to manually fetch the latest commands for registry-installed libraries:

```bash
my library sync
```
