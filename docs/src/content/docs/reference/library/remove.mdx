---
title: my library uninstall
description: Remove a registry-installed library.
---

```bash
my library uninstall <name>
```

Removes a library that was installed from the registry.

## Arguments

| Argument | Description |
|----------|-------------|
| `name` | Name of the library to uninstall (required) |

## Examples

```bash
my library uninstall kubernetes
```

## Behavior

- Removes the library and its commands from the local installation
- Library commands are no longer available

:::note
To remove a git-backed source, use [`my source remove`](/reference/source/remove/) instead.
:::
