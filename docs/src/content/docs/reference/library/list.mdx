---
title: my library list
description: List all installed libraries.
---

```bash
my library list [flags]
```

Lists all installed libraries from all sources (both git-backed sources and registry-installed libraries) with their names, source type, and available commands.

## Flags

| Flag | Description |
|------|-------------|
| `--json` | Output as JSON |

## Examples

```bash
# List all libraries
my library list

# Output as JSON
my library list --json
```
