---
title: my library list
description: List all installed libraries.
---

```bash
my library list [flags]
```

Lists all installed libraries with their names, URLs, and available commands.

## Flags

| Flag | Description |
|------|-------------|
| `--json` | Output as JSON |

## Examples

```bash
# List all libraries
my library list

# Output as JSON
my library list --json
```
