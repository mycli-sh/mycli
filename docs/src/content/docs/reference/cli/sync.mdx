title: my cli sync

---

```bash
my cli sync
```

Fetches the latest catalog from the API and caches specs locally. Also updates shelves via `git pull`.

## Behavior

- Downloads the command catalog from the API server
- Uses ETag-based caching for efficient bandwidth usage
- Caches specs locally under `~/.my/cache/`
- Pulls latest changes from all added shelf repos
- Works without login — skips API sync and only updates shelves

## Examples

```bash
# Sync everything
my cli sync

# After login, sync includes API catalog
my cli login
my cli sync
```

## Requires

No authentication required. API catalog sync is skipped when not logged in.
