---

```bash
my cli update
```

Updates mycli to the latest version.

- If installed via **Homebrew**, prints the `brew upgrade` command to run.
- Otherwise, downloads and runs the install script to fetch the latest release.

## Automatic update checks

mycli checks for new versions in the background (once every hour). If a newer version is available, a notification is printed after the command finishes.
