---
title: my cli status
description: Show API URL, login state, and sync status.
---

```bash
my cli status
```

Displays the current CLI status including API connection and sync information.

## Output

- API URL
- Login state (authenticated or not)
- Last sync time
- Number of cached commands

## Examples

```bash
my cli status
# API:      https://api.mycli.sh
# Logged in: yes (user@example.com)
# Last sync: 2 minutes ago
# Commands:  12 cached
```
