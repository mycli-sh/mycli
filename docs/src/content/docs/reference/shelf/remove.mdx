---
title: my shelf remove
description: Remove an added shelf.
---

```bash
my shelf remove <name>
```

Removes a shelf from the registry and deletes the cloned repository.

## Arguments

| Argument | Description |
|----------|-------------|
| `name` | Name of the shelf to remove (required) |

## Examples

```bash
my shelf remove my-shelf
```

## Behavior

- Removes the shelf entry from `~/.my/shelves/shelves.json`
- Deletes the cloned repository from `~/.my/shelves/repos/`
- Shelf library commands are no longer available
