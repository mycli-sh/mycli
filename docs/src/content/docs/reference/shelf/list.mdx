---
title: my shelf list
description: List all added shelves.
---

```bash
my shelf list [flags]
```

Lists all registered shelves with their names, URLs, and libraries.

## Flags

| Flag | Description |
|------|-------------|
| `--json` | Output as JSON |

## Examples

```bash
# List all shelves
my shelf list

# Output as JSON
my shelf list --json
```
