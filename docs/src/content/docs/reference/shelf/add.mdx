---
title: my shelf add
description: Add a shelf from a git repository.
---

```bash
my shelf add <url> [flags]
```

Clones a shelf repository, validates the manifest and command specs, and registers it.

## Arguments

| Argument | Description |
|----------|-------------|
| `url` | Git repository URL (required) |

## Flags

| Flag | Description |
|------|-------------|
| `--ref` | Git branch or tag to checkout |
| `--name` | Alias for the shelf (defaults to repo name) |

## Examples

```bash
# Add a shelf
my shelf add https://github.com/user/my-shelf.git

# Add with a specific branch
my shelf add https://github.com/user/my-shelf.git --ref v1.0

# Add with a custom name
my shelf add https://github.com/user/my-shelf.git --name tools
```

## Behavior

1. Clones the repository to `~/.my/shelves/repos/`
2. Reads the shelf manifest (`shelf.yaml`, `shelf.yml`, or `shelf.json`)
3. Validates all command specs in each library directory
4. Registers the shelf in `~/.my/shelves/shelves.json`
5. Commands become available as `my <library> <slug>`
