---
title: my shelf update
description: Update one or all shelves.
---

```bash
my shelf update [name]
```

Pulls the latest changes from shelf repositories via `git pull`.

## Arguments

| Argument | Description |
|----------|-------------|
| `name` | Optional. Name of a specific shelf to update. If omitted, updates all shelves. |

## Examples

```bash
# Update all shelves
my shelf update

# Update a specific shelf
my shelf update my-tools
```
